# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-27

### Added
- Product CRUD (list, get, create, update, delete) with variant and option support
- Page CRUD (list, get, create, update, delete) with SEO metafields
- Collection read (list, get)
- Menu read (list, get) with nested item hierarchy
- File list with type filtering
- Theme list
- Shop metadata
- Raw GraphQL escape hatch (`gql`) with inline, file, and stdin support
- Config show with masked credentials
- JSON output mode (`--json`) for all commands
- Client Credentials Grant authentication with token caching
- Biome for linting and formatting
- GitHub Actions CI
